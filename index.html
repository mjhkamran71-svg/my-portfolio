CSS Cascading.

CSS cascading refers to the way the browser decides which css rules target the same html element.

The word "cascading" literally means styles flow down and are applied based on priority rules.

Combining CSS selector.

Combining Selectors means targeting elements using more than one selector together.

CSS Positioning.

  • Controls how an element is placed on a webpage.
  • The position property works together with properties like top, bottom, left, and right.

CSS display.

  • The display property in css controls how an HTML element is shown on a webpage.
  • for example,whether it behave like a block, inline text or something else.

CSS float.

  • In CSS the float property is used to position an element to the left or right side of its container, allowing text or inline elements to wrap arround it.
  • It wa originally designed for layouts Like wrapping texts arround images.

Responsive Web Desigh.

  • Responsive web design is a web design approach that makes a website automatically adjust its layout, images and content to look good on different screen sizes- such as phones, tablets, laptops and desktops.

Media queries.

  • Media queries are a feature in css used to make websites. responsive.
  • They allow your layout or styles to change depending on things like screen size, device type or orientation.

Flexbox

Flexbox is a css layout system that helps you easily design and align elements in a row or column- even when their sizes are unknown or dynamic.

  • flexbox works with:
    • flex container (parent).
    • flex item (children).

  2. Flex Direction.

Grid
